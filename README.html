卒業論文クラス ampbt.cls の説明書 （2023/12/20）

はじめに

ampbt.cls は数理工学コースの卒業論文を書くためのクラスファイルです． ampbt.cls を用いると，様式が指定されている表紙，摘要，目次が必要項目を入力するだけで適切に生成されます．また，サンプルファイルで論文の形式の雛形も与えています．

準備

ダウンロードしてきた ZIP ファイルを作業用のディレクトリ（卒業論文の原稿ファイルを置くフォルダ）で解凍します．以下のファイルが生成されます．

  • README.html：このファイル．
  • ampbt.cls：クラスファイル本体．
  • ampbt_sample.tex, ampbt_sample.pdf：サンプルソースとその出力．
  • ampbt_figure.pdf, ampbt_figure.tex：サンプルファイル中に挿入されている図とそのソース．

文字コードは全て UTF-8 に統一されています．エディタ等で文字化けが生じた場合は UTF-8 に対応しているかどうか確認して下さい．

使い方

ampbt_sample.tex を同じディレクトリ内にコピーし，適当な名前に変更します．エディタで開いてプリアンブルに題目等の項目をコメントの説明に従って記入します．その後，本文の執筆にとりかかって下さい．

タイプセットは通常通り platex + dvipdfmx を用いればよいです．万一 platex が UTF-8 のコードをうまく認識しない場合は -kanji=utf8 を platex のオプションに指定して下さい：

    platex -kanji=utf8 thesis.tex

出力は，表紙，摘要，目次，本文，提出用摘要の順となります．PandAに締切日時までに提出して下さい．提出する前に題目や各人の氏名，年月日等が合っているかどうか必ず確認すること．

注意点として，本文を両面印刷場合，本文以外については片面印刷にすることができます．クラスオプションに twoside を指定すると，これらのページの後ろに空白ページが適切に挿入され，両面印刷設定でそのまま印刷できるようになります．

よくあるかもしれない質問と回答

一般的なエディタや LaTeX の使い方については各自で書籍や Web ページ，優秀な友人や先輩にあたって下さい．処理中にエラーが出た場合は，まずはエラーメッセージを読んで何が起きているのか理解することを試みましょう（英語だからと言って思考停止しない）．

Q：末尾の提出用摘要が出力されないように（もしくはこれらだけが出力されるように）したいです．
A：サンプルの \begin{document} の直前にある「出力の制御」のところをいじって下さい．
Q：フォントサイズを変えたいです．
A：デフォルトでは 11pt になっていますが，通常通りクラスオプションに 12pt と指定すれば大きくできます．10pt と小さくすることもできますが，小さな文字は読みにくいので推奨しません．
Q：\chapter コマンドを使うと ! Undefined control sequence. と出ます．
A：デフォルトでは jsarticle をベースに用いるため，\chapter は未定義でエラーが出ます．\chapter を使いたい場合はクラスオプションに chapter を指定すると jsbook[report] をベースに用いるようになります．
Q：\vspace{...} などで指定した寸法よりも若干大きなスペースが入っているようですが．
A：これは jsclasses をベースに用いているために起きる現象です．フォントサイズが 11pt の場合，指定した寸法を1.1倍した値になってしまいます．これを回避するためには，単位 xx の代わりに truexx を用いて下さい（2truemm など）．
Q：レイアウトを変えるために geometry パッケージを読み込むと ! LaTeX Error: Option clash for package geometry. と出ます．
A： ampbt.cls 内で既に geometry を読み込んでいるために，プリアンブルにオプション付きで geometry を読み込むと option clash となります．\begin{document} の直前に書いてあるように，\geometry コマンドを使って下さい．
Q：platex に -kanji=utf8 オプションをつけましたが platex: unrecognized option '-kanji=utf8' と出ます．
Q：\numexpr という書いた覚えのない命令に対して ! Undefined control sequence. と出ます．
A：どうやら pTeX のバージョンが古すぎるようです．最新の TeX システム（TeX Live など）を導入しなおして下さい．
Q：uplatex を使いたいのですが，! LaTeX Error: Font family `JY1+mc' unknown. と出ます．
A：クラスオプションに uplatex を指定して下さい．
Q：xelatex や lualatex を使いたいのですが，! LaTeX Error: This file needs format `pLaTeX2e' but this is `LaTeX2e'. と出ます．
A：jsclasses をベースに使っているので，platex, uplatex 以外ではエラーが出ます．今のところ xelatex や lualatex に対応する予定はありませんが，どうしても使いたい場合は自身で ampbt.cls に手を入れて下さい．
